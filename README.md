# fruit-classification-mobilenetv2-flask
Implementasi Transfer Learning MobileNetV2 untuk Klasifikasi Jenis Buah Berbasis Website Menggunakan Flask. Project ini menggunakan deep learning untuk mengklasifikasikan gambar buah (Apple, Banana, Grape, Mango, Strawberry) dan menampilkan hasil prediksi secara real-time melalui web interface.
